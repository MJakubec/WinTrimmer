WinTrimmer

Tool for dynamic modification of the appearance of the Windows user interface by group of customizable templates.

Download this application as a .zip file

Motivation

There are many windows and dialog boxes of the Windows operating system that are designed rather poorly. Instead to be continuously annoyed with it, one possible way how to improve them is to create an application that will be able to customize layout of selected parts of the Windows user interface. This project is aimed at creation of such application equipped with a comprehensive set of templates describing the whole process of the user interface identification and customization.
